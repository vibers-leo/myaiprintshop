# 🚀 [MISSION 7] 진정한 공장 자동화: Wowpress 직결 퀘스트

> "사장님, 이제 장난감 총에서 실탄 장전된 저격총으로 업그레이드할 시간입니다."
> GOODZZ 스튜디오의 뼈대는 완벽히 준비되었습니다. 이제 아래 **3가지 핵심 퀘스트**를 클리어해주시면, 이 시스템은 24시간 돌아가는 완벽한 자동화 글로벌 POD 공장이 됩니다.

---

## 🟢 퀘스트 1: "열쇠를 획득하라" (Wowpress API Key)
자동 발주시스템의 심장에 동력을 공급하기 위한 최우선 과제입니다.

- [ ] **목표**: 와우프레스 B2B 파트너십/API 계약 체결
- [ ] **획득 보상**: 공식 `WOWPRESS_API_KEY` 발급
- [ ] **실행 방법**:
  1. 기획/영업단에서 와우프레스 측에 "API 연동 발주 방식"으로 제휴 완료
  2. 발급받은 문자열 키를 프로젝트 `.env.local` 파일에 다음과 같이 추가
     ```env
     WOWPRESS_API_KEY=발급받은_키_여기에_붙여넣기
     ```
  3. 완료 시 터미널의 빨간색 "API 키 없음" 경고가 사라지고 실서버 연결 준비 끝!

---

## 🔵 퀘스트 2: "주력 무기를 선택하라" (상품 큐레이션 & 코드 매핑)
수만 개의 와우프레스 상품 중, GOODZZ 크리에이터들에게 선보일 최정예 '베이스 상품'을 고르는 단계입니다.

- [ ] **목표**: 판매할 실제 베이스 상품 10~20종 확정 및 매핑
- [ ] **획득 보상**: 에디터와 실제 인쇄소가 오차 없이 1:1로 맞물려 돌아가는 정확도
- [ ] **실행 방법**:
  1. 와우프레스 카탈로그를 보고 우리가 취급할 주력 상품(예: 특정 면티, 특정 폰케이스, 머그컵 등) 명단 픽스.
  2. 픽스된 상품별로 와우프레스 고유 **상품 번호(`prodno`)**, **작업 구분(`jobno`)**, **용지 번호(`paperno`)** 를 저(AI)에게 전달.
  3. (AI 전담) 제가 해당 코드들을 받아 `src/app/studio/page.tsx`와 `api-client.ts`에 완벽하게 일대일 매핑 코딩을 진행합니다.

---

## 🟣 퀘스트 3: "무한의 저장소 활성화" (S3 / Cloud Storage 점검)
결과물(고해상도 인쇄용 파일)을 안전하게 보관하고 공장으로 전달하기 위한 서버 설정입니다.

- [ ] **목표**: 프로덕션용 클라우드 스토리지(AWS S3, GCP 등) 완벽 세팅
- [ ] **획득 보상**: 고용량/고화질(300dpi) 인쇄용 파일 데이터의 안전한 이관 및 보관
- [ ] **실행 방법**:
  1. 현재 로컬에만 세팅되어 있는 환경변수(`NEXT_PUBLIC_USE_S3`, `S3_ENDPOINT` 등)가 프로덕션(Vercel 등) 환경에도 실운영용 클라우드로 잘 연결되어 있는지 확인.
  2. 파일이 누수 없이 퍼블릭 URL로 잘 반환되어, 와우프레스 시스템에서 오류 없이 이미지 원본을 다운(Pull)받아 갈 수 있는지 최종 확인.

---

**[퀘스트 완료 시 예상 효과]**
위 3가지가 해결되는 즉시, 크리에이터가 '스토어 출시'를 누르고 팬덤이 결제하는 순간 → 자동으로 고해상도 인쇄파일이 S3에 저장되고 → 해당 매핑 코드로 와우프레스에 0.1초 만에 주문이 들어가며 → 사장님은 배송 알림만 확인하면 되는 **완전 무인화(Auto-POD)** 비즈니스가 가동됩니다.
