## 1. 브랜드 개요

### 브랜드명
- **영문**: GOODZZ (goodzz)
- **한글**: 굿쯔
- **도메인**: goodzz.co.kr (구매 예정)
- **패키지명**: @vibers/goodzz

### 브랜드 네이밍 의도
"Goods(굿즈)" + "zz(잠자는 아이디어를 깨운다)" → 소상공인의 브랜드 굿즈를 AI로 쉽게 만들어주는 플랫폼. "쯔"의 발음이 주는 친근하고 귀여운 느낌으로 접근 장벽을 낮춤.

### 슬로건 후보
1. **"내 브랜드 굿즈, AI로 뚝딱"** — 직관적, 소상공인 어필
2. **"브랜드를 만지다"** — 감성적, 프리미엄 뉘앙스
3. **"사장님의 굿즈 공장"** — B2B 느낌, 실용적

## 2. 타겟 분석

### Primary: 소상공인 / 자영업자
| 항목 | 내용 |
|------|------|
| 페르소나 | 카페/베이커리/플라워샵/네일샵/펫샵 사장님 (25~45세) |
| 핵심 니즈 | 명함, 스티커, 포장재, 쇼핑백, 에코백 등 브랜드 굿즈 제작 |
| 현재 Pain | 디자인 외주 비용 높음(50~200만원), 소량 주문 거부, 시안 수정 커뮤니케이션 스트레스 |
| 기대 가치 | AI로 직접 디자인 → 소량 가능 → 저렴하고 빠르게 |
| 결제 의향 | 건당 1~5만원 (명함 500장 2만원, 스티커 100장 3만원 수준) |

### Secondary: 소규모 브랜드 / 팝업 운영자
| 항목 | 내용 |
|------|------|
| 페르소나 | 핸드메이드 브랜드, 팝업스토어 운영자, 마켓 셀러 |
| 핵심 니즈 | 패키징(박스, 택, 스티커), 홍보물(전단, 배너), 굿즈(키링, 엽서) |
| 현재 Pain | 매번 디자이너에게 의뢰 → 느리고 비쌈, 일관된 브랜드 아이덴티티 유지 어려움 |
| 기대 가치 | 브랜드 컬러/로고 등록 → 일관된 디자인 자동 생성 |

### Anti-Target (제외)
- 대기업/중견기업 (자체 디자인팀 보유)
- 순수 개인 크리에이터 (기존 myaiprintshop 타겟 — 굿즈 판매보다 자기 표현)
- 가격 민감도가 극도로 높은 대량 인쇄 수요 (기존 인쇄소 직거래가 유리)

## 3. 포지셔닝

### 포지셔닝 맵

```
       고가
        |
  디자인 에이전시  ───  캔바 프로
        |                    |
  낮은 맞춤화 ─────────────── 높은 맞춤화
        |                    |
  일반 인쇄소  ───  ★ GOODZZ
        |
       저가
```

### 핵심 차별화
1. **AI 디자인 + 인쇄 원스톱**: 캔바는 디자인만, 인쇄소는 인쇄만 → 굿쯔는 둘 다
2. **소량 특화**: 100장부터 주문 가능 (기존 인쇄소 500~1000장 최소)
3. **브랜드 키트**: 로고/컬러/폰트 등록하면 모든 굿즈에 자동 적용
4. **소상공인 맞춤 카테고리**: 명함, 스티커, 포장재, 쇼핑백, 에코백, 전단지, 메뉴판

### Value Proposition
> **"디자이너 없이, 소량으로, 내 브랜드 굿즈를 만드세요"**

## 4. 톤 & 매너

### 브랜드 성격
- **친근함**: 사장님들이 편하게 느끼는 톤 ("어렵지 않아요!")
- **신뢰감**: 인쇄 품질/배송에 대한 확신 ("실물이 이쁘더라")
- **실용적**: 꾸밈 없이 핵심만 ("3분이면 완성")

### 커뮤니케이션 톤
| 상황 | 톤 | 예시 |
|------|-----|------|
| 온보딩 | 격려 | "사장님, 첫 굿즈 만들어볼까요?" |
| AI 생성 | 기대감 | "사장님 브랜드에 딱 맞는 디자인을 만들고 있어요" |
| 결제 | 안심 | "소량이라 부담 없어요. 마음에 안 들면 전액 환불" |
| 배송 | 신뢰 | "인쇄 완료! 내일 도착 예정이에요" |

### 금지 표현
- "크리에이터", "아티스트" (타겟이 아님)
- "혁신적인", "최첨단 AI" (기술 강조보다 결과 강조)
- 과도한 영어 ("커스터마이즈" → "맞춤 제작")

## 5. 비주얼 가이드

### 컬러 시스템
| 용도 | 컬러 | HEX | 의미 |
|------|-------|-----|------|
| Primary | 따뜻한 코랄 | #FF6B6B | 친근함, 에너지 |
| Secondary | 소프트 네이비 | #2D3A4A | 신뢰, 전문성 |
| Accent | 민트 그린 | #4ECDC4 | 신선함, 완성 |
| Background | 크림 화이트 | #FFF9F5 | 따뜻한 배경 |
| Surface | 라이트 그레이 | #F5F0EB | 카드/섹션 구분 |
| Text Primary | 다크 브라운 | #2C2420 | 부드러운 가독성 |
| Text Secondary | 웜 그레이 | #8B7E74 | 보조 텍스트 |

### 타이포그래피
- **한글**: Pretendard (본문), 노토산스 (대체)
- **영문**: Geist (UI), Inter (본문)
- **강조**: Pretendard Bold, 크기 차이로 위계 표현
- 모든 카피는 한글 우선, 영문은 보조

### UI 원칙
1. **둥글게**: border-radius 12px+ (친근함)
2. **여백 넉넉하게**: 소상공인은 디지털 경험이 다양 → 터치 영역 크게
3. **실물 미리보기**: AI 생성 후 실제 인쇄물 목업 필수
4. **단계별 진행**: 위자드 형태 (카테고리 선택 → AI 디자인 → 수량/옵션 → 결제)

## 7. GTM (Go-To-Market) 전략

### Beachhead: 카페 사장님
- **이유**: 명함+스티커+포장재 세트 니즈가 명확, SNS 활발, 입소문 강함
- **초기 목표**: 카페 사장님 50명 유치 (1개월)

### 채널 전략
1. **인스타그램**: 카페 사장님 DM + "내 카페 굿즈 무료 체험" 이벤트
2. **네이버 카페**: 자영업자/카페 커뮤니티 후기 마케팅
3. **로컬 파트너십**: 지역 카페 협회, 소상공인진흥공단 연계
4. **크로스셀**: 와우프레스(myaiprintshop.co.kr) 기존 트래픽에서 유입

### 가격 전략
- **무료 체험**: AI 디자인 생성 3회 무료 (회원가입 유도)
- **첫 주문 할인**: 30% 할인 (전환율 부스팅)
- **세트 할인**: 명함+스티커+전단지 세트 20% 할인
